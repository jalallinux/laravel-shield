A HTTP basic auth middleware for Laravel

## Installation

You can install the package via composer:

```bash
composer require jalallinux/laravel-shield
```

## Usage

```php
$skeleton = new JalalLinuX\Shield();
echo $skeleton->echoPhrase('Hello, JalalLinuX!');
```

## Testing

```bash
composer test
```
